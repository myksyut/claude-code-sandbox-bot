# Claude Code Sandbox Skills

このファイルはサンドボックス内でClaude Codeが起動時に読み込む設定です。

## ユーザーへの質問

ユーザーに確認が必要な場合は、以下のコマンドを使用してください:

```bash
python /tools/ask_user.py "質問内容"
```

回答は標準出力に返されます。

### 使用例

```bash
# 単純な質問
python /tools/ask_user.py "このファイルを削除してもよいですか？"

# 複数の選択肢がある質問
python /tools/ask_user.py "どのブランチにプッシュしますか？ (main / develop / feature)"
```

### 注意事項

- 質問はSlackスレッドに転送されます
- 回答を待機する間、処理は一時停止します
- タイムアウトは10分です（タイムアウト後はエラーになります）
- 緊急でない限り、ユーザーの回答を待ってから次の処理に進んでください
